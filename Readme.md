# 🌮 The Little Taco Shop Website

A responsive restaurant website built using HTML, CSS, and JavaScript.

## 🚀 Features
- Responsive design (mobile & desktop)
- Navigation menu with multiple pages
- Styled menu section using CSS Grid
- Smooth scrolling and modern UI
- Accessible and semantic HTML structure

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Variables)
- JavaScript



## 💻 How to Run Locally

Clone the repository:  git clone https://github.com/Mario-Culica/The-Little-Taco-Shop-Project.git
